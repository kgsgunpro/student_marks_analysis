# Student Performance Analysis

This notebook performs an exploratory data analysis on a generated dataset of student performance. The analysis includes calculating various metrics, identifying top performers, and visualizing score distributions.

## Data Generation
- A dataset of 10 students with random scores for Math, Science, and English was generated.

## Data Preprocessing and Feature Engineering
- **Total Score:** Calculated as the sum of Math, Science, and English scores for each student.
- **Average Marks:** Calculated as the average of Math, Science, and English scores for each student.

## Descriptive Statistics
- **Average Scores by Subject:** Calculated the average scores for Math, Science, and English.
- **Standard Deviation of Scores by Subject:** Calculated the standard deviation for Math, Science, and English scores.
- **Summary Statistics:** A comprehensive summary statistics table for the entire DataFrame was generated.

## Visualizations
- **Average Scores by Subject Bar Chart:** Visualized the average scores across the three subjects.
- **Total Scores per Student Bar Chart:** Displayed the total scores for each student.
- **Average Marks per Student Bar Chart:** Displayed the average marks for each student.

## Key Findings
- Identified the top 3 students based on their average marks.
